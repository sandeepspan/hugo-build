### Notes from Prerequisits

* OpenShift console URL is, 
```
     https://10.2.2.2:8443/console
```
* Configured users are (<username>/<password>):
```
    - admin/admin
    - user/user
```    
* OpenShift CLI can be run as,
``` 
    $ oc login https://10.2.2.2:8443  
```
